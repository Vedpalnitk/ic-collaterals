About

A working library — built incrementally as each new collateral stress-tests the design system. Artifacts and system co-evolve.

Built with Fraunces, IBM Plex Sans, Newsreader, and JetBrains Mono. The visual vocabulary — the seam, chapter slabs, eyebrow bars, drop caps, named callouts — is documented in the design reference. Current identity is the third iteration; earlier "consulting bland" attempts were rejected for the editorial-broadsheet feel.
