About

This is a working library — built incrementally as each new collateral type stress-tests the design system and surfaces what needs to be added to it. The artifacts and the system co-evolve.

Built with Fraunces (display, with optical-size and WONK axes), IBM Plex Sans (technical body), Newsreader (editorial body), and JetBrains Mono (utility). The visual vocabulary — the seam, chapter slabs, eyebrow span bars, drop caps, named callouts — is documented in the design reference.

The current visual identity is the third iteration; earlier attempts at "consulting bland" were rejected in favor of the editorial-broadsheet feel that runs across the library now.
